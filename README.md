# Online Shoppers Purchase Prediction (Deployment Project)

## 📌 Project Overview
This project predicts whether a user will make a purchase on an e-commerce website using Machine Learning and deploys the model using Flask on AWS EC2.

## 🚀 Features
- Logistic Regression model
- SMOTE for class imbalance
- Flask web app for real-time prediction
- AWS EC2 deployment

## 🛠️ Tech Stack
Python, Pandas, Scikit-learn, Flask, AWS EC2, HTML/CSS

## 📂 Files Included
- app.py → Flask application
- model.pkl → Trained ML model
- dataset.xlsx → Dataset
- model_training.ipynb → Model building
- index.html → Frontend UI
- project.pptx → Presentation

## ▶️ How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python app.py`
